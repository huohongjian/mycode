一、运行命令：$dgrid.run({}),参数说明如下：

  • 方法
  • 参数
  • 说明
  • $dgrid.run({})
  • id :''
  • 指定将dgrid添加到页面元素的id
  • align :[]
  • 各列的对齐方式
    • 　　　　
  • width :[]
  • 各列的宽度，单位：px
  • 　　　　
  • headh :[[]]
  • 横表头
  • 　　
  • headv :[[]]
  • 纵表头
  • 　
  • data :[[]]
  • 表体数据
  • rtype :[]
  • 各行数据类型
  • ctype :[]
  • 各列数据类型
  • rdecimal:[]
  • 各行小数位
  • cdecimal:[]
  • 各列小数位
  • items :{}
  • 单元格输入时的选择项，以行或列设置
  • height : n
  • 表体高度(px)
  • heightOff: n
  • 表体top偏离度(px)
  • cellHeight: n
  • 单元格高度
  • cellWidth: n
  • 单元格宽度（默认值）
  • rfloat: boolean
  • 表体是否可行浮动
  • cfloat: boolean
  • 表体是否可列浮动
  • editable:true
  • 表体是否可编辑
  • enterDown:false
  • 回车是否纵向移动
  • auditings:[]
  • 审核公式
  • operations:[]
  • 运算公式
