四、$table对象公有方法

  • 用法
  • 返回值
  • 说明
  • $tbl().init(t)
  • $table
  • 初始化$table对象，等同于$tbl(t)
  • $tbl().fit()
  • $table
  • 规整table元素，使各行的列数相等
  • $tbl().appendTo(a)
  • Bealoon
  • 将table元素追加到指定的元素下，a为id或obj
  • $tbl().insRows(idx, args)
  • Bealoon
  • 在指定位置插入行，省略idx或idx=-1时追加行
  • $tbl().insCols(idx, args)
  • Bealoon
  • 在指定位置插入列，省略idx或idx=-1时追加列
  • $tbl().getData(x, y, X, Y)
  • 2d-Array
  • 获取指定区域数据，省略参数时，获取全部数据
  • $tbl().getRows(args)
  • Array
  • 获取指定行数据，省略参数时为全部数据，指定1行时，返回值为1d-Array
  • $tbl().getCols(args)
  • Array
  • 获取指定列数据，省略参数时为全部数据，指定1列时，返回值为1d-Array
  • $tbl().getCell(x, y)
  • String
  • 获取指定单元格数据
  • $tbl().setData(df)
  • Bealoon
  • 将DataFrame(df) which is 2d-Array值赋予table元素
  • $tbl().fmtData(digital)
  • Bealoon
  • 格式化数据，digital为小数位
  • $tbl().mergeCells(b)
  • Bealoon
  • 合并空单元格，参数为是否仅纵向合并最后一行为空的单元格
